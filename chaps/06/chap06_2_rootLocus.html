The Root Locus Method for Finding the Interval of Absolute Stability

6.2. The Root Locus Method for Finding the Interval of Absolute Stability#

This method consists of repeatedly solving the characteristic polynomial for various values of \(h\lambda\) in the neighbourhood of the origin. Note that for simple linear and quadratic cases the resulting characteristic polynomial can be solved and a table or plots of values of \(\,h\lambda\,\) against roots \(|z_1|\) and \(|z_2|\) allows us to determine the interval of absolute stability. Any numerical method such as Newton Raphson iteration, or mathematics software packages like Matlab may be used for the approximate solution of the characteristic polynomial.

Example 6.1

Find the interval of absolute stability of the two-step explicit AB method,

\[ \begin{aligned} y_{j+1} ~=~ y_j + \frac{h}{2}(3f_j - f_{j-1}) \end{aligned} \]

when applied to \(y' = \lambda y\).

Solution

The characteristic equation is

\[ \begin{aligned} L(z,h\lambda) ~=~ z^2 - \left(1 + \frac{3}{2}h\lambda\right)z + \frac{1}{2}h\lambda ~=~ 0 \end{aligned} \]

and its roots are

\[\begin{split} \begin{aligned} z_1 ~&=~ \frac{1}{2} \left( 1 + \frac{3}{2}h\lambda + \sqrt{1 + h\lambda + \frac{9}{4}\left(h\lambda\right)^2} \right) \\[1ex] z_2 ~&=~ \frac{1}{2} \left( 1 + \frac{3}{2}h\lambda - \sqrt{1 + h\lambda + \frac{9}{4}\left(h\lambda\right)^2} \right) \end{aligned} \end{split}\]

By definition the interval of absolute stability is the set of \(h\lambda\) for which

\[ \begin{aligned} \left| z_1(h\lambda) \right| < 1, \qquad \left| z_2(h\lambda) \right| < 1\,. \end{aligned} \]

For \(\lambda\) real, the acceptable values of \(\,h\lambda\,\) are \(-1 \leq h\lambda \leq 0\,\); i.e. the method is stable inside the interval \(\,[-1,0]\,\).

This result implies that if the 2-step AB method is used to solve the differential equation \(\,y' = -100y\), with \(\lambda = -100\), and \(\,h\lambda > -1\), then \(\,h < \dfrac{1}{100}\) for the method to be absolutely stable.

Example 6.2

Find the interval of absolute stability for the implicit, two-step, Adams method

\[ \begin{aligned} y_{j+1} ~=~ y_j + \frac{h}{12}(5f_{j+1} + 8f_j - f_{j-1}) \end{aligned} \]

when applied to \(\,y' = \lambda y\).

Solution

The characteristic equation is

\[ \begin{aligned} L(z,h\lambda) ~=~ \left(1 - \frac{5}{12}h\lambda\right)z^2 - \left(1 + \frac{8}{12}h\lambda\right)z + \frac{1}{12}h\lambda ~=~ 0 \end{aligned} \]

Let \(a=\dfrac{h\lambda}{12} \) and its roots are

\[ \begin{aligned} z_{1,2} = \frac{1 + 8a \pm\sqrt{\left(1 + 8a\right)^2 - 4a\left(1 - 5a\right)}}{2(1 - 5a)} \end{aligned} \]
